## 🔄 Vital 循环

### 健康→财富循环
```
好睡眠 → 高能量 → 高产出 → 高收入
      ↑                          ↓
      ←  财务安全 ← 储蓄投资  ←
```

### 财富→健康循环
```
财务稳定 → 降低压力 → 更好睡眠 → 更好健康
      ↑                            ↓
      ←   更多选择 ← 投资健康  ←
```

**正循环强化**:
- 用收入的10%投资健康
- 用健康的能量创造财富

**负循环警报**:
- [ ] 为了赚钱牺牲睡眠
- [ ] 为了省钱牺牲健康
- [ ] 压力导致冲动消费
