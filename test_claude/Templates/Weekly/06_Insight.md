## ⚡ 快速检查

### 质量标准
- [ ] 一句话测试: 能向朋友用30秒说清楚
- [ ] 提取测试: 不看笔记能回忆出核心
- [ ] 可操作测试: 能立即想到1个应用场景
- [ ] 独立测试: 没有上下文也能理解
- [ ] 证伪测试: 有明确的证伪条件

### 创建后必做
- [ ] 设置24小时后的第一次复习提醒
- [ ] 选择空间锚点（可选但推荐）
- [ ] 创建荒诞的视觉化故事
- [ ] 至少连接1个应用场景

### 维护规则
- **更新触发**: 确信度变化>20%时更新
- **归档标准**: 3个月未调用 + 确信度未提升 → Archive
- **毕业标准**: 连续3次复习为"easy" + 确信度>90% → 已内化
