## 🧠 主动回忆测试

> [!WARNING] 重要规则
> **不要先看原笔记！** 主动回忆的核心是"努力提取"，这个过程本身会强化记忆。先看答案会让大脑产生"我记得"的错觉，实际上没有形成长期记忆。

### 第一关: 核心提取 (必答)
**不看原笔记，用一句话说出核心内容:**
```
{{你的答案}}
```

**难度自评**: 
- [ ] 轻松: 几乎不用思考就想起来了
- [ ] 中等: 需要思考10-30秒
- [ ] 困难: 思考很久，只记得模糊印象
- [ ] 失败: 完全想不起来
