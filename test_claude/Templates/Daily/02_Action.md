## ⚡ 快速参考

### 检查清单
**开始前**:
- [ ] 目标明确可验证
- [ ] 第一步可立即执行  
- [ ] 资源已识别
- [ ] 风险已评估

**进行中**:
- [ ] 每周至少更新一次状态
- [ ] 遇到阻塞立即记录
- [ ] 偏离计划及时调整

**完成后**:
- [ ] 创建Loop复盘
- [ ] 更新相关笔记
- [ ] 归档或删除

### 使用提示
- **更新频率**: 每周至少一次
- **时间投入**: 规划15分钟，追踪5分钟/周
- **完成强制**: 必须创建Loop

### 质量标准
- 目标可在30秒内说清楚
- 第一步可在1小时内完成
- 完成标准客观可验证
