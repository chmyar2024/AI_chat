### 2. 边界厘清 (5组)

✅ **Seed vs Intuition**  
- Seed = 外部信号 (文章/对话)
- Intuition = 内在信号 ("我就是知道")

✅ **Create vs Play**  
- Create = 有产出目标
- Play = 纯粹探索/恢复能量

✅ **Book vs Insight**  
- Book = 容器 (只保留链接)
- Insight = 提取物 (完整内容)

✅ **Persona vs Organization vs Community**  
- Persona = 个人
- Organization = 正式组织
- Community = 非正式群体

✅ **Skill vs System**  
- Skill = 会做 (需要练习)
- System = 知道 (需要理解)
